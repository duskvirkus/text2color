Colophon
Text2Color is an open-source site (see contributers). It was built using p5.js, p5.js dom, tensorflow.js, bootstrap, jquery, and popper.js libraries. The logo by Fi Graham was designed using Molle font by Elena Albertoni and Adobe Illustrator. The font used is Open Sans by Steve Matteson. The site is hosted through GitHub Pages.
